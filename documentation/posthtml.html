Simple component connection:

Example (with Alias)

<include src="@c/header/_header.html"></include>

or (without alias)

<include src="./src/html/components/header/_header.html"></include>

Connecting component with parameters:

Example (parameters are transmitted in attribute locals='{}' In the object, the object must be in single paws.)

<include
  src="@c/card/_card.html"
  locals='{"title": "Title", "description": "Опис", "img": "@img/1/1.jpg"}'
</include>

можна відформатувати

<include
    src="@c/card/_card.html"
    locals='{{
        "title": "Title",
        "description": "Description",
        "img": "src/assets/img/1/1.jpg"
    }}'>
</include>

Using parameters transmitted through attribute locals in the component.

Example

<div class="block">
    <h2>{{title}}</h2>
    <p>{{description}}</p>
    <img src="{{img}}" alt="{{title}}">
</div>
