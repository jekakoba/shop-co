Using a conditional expression

*The parameters used in the conditions (here isctive and Title) must be transmitted and must either have a value or be empty !!!

Examples of use in the component in which the parameters are transmitted title and isActive.

<div class="item {{ isActive ? 'active' : 'hidden' }}">
    <h2>{{ title ? title : 'There is no headline' }}</h2>
</div>
<div class="item {{ isActive ? 'active' : 'hidden' }}">
    {{title? <h2>{{title}}</h2> : ''}}
</div>

Using if elseif and else:

Example Use in the component in which the parameter is transmitted title.

<div class="item">
    <h2>
        <if condition="title === 'bar'">Title: BAR</if>
        <elseif condition="title === 'wow'">Title: WOW</elseif>
        <else>{{title}}</else>
    </h2>
</h2>
