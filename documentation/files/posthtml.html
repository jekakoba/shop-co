HTML-компоненти

Цей документ описує підключення компонентів, передачу параметрів, вивід контенту з json файлу, цикли, умови та інших умовних виразів.

Головна Структура Умови та умовні вирази Цикли та json Налаштування

Просте підключення компонента:

Приклад

<include src="./src/html/components/header/_header.html"></include>

Підключення компонента з параметрами:

Приклад (параметри передаються в атрибуті locals='{}' в об'єкті, при при цьому об'єкт має бути в одинарних лапках.)

<include
  src="./src/html/components/card/_card.html"
  locals='{"title": "Заголовок", "description": "Опис", "img": "src/assets/img/1/1.jpg"}'
</include>

можна відформатувати

<include
    src="./src/html/components/card/_card.html"
    locals='{{
        "title": "Заголовок",
        "description": "Опис",
        "img": "src/assets/img/1/1.jpg"
    }}'>
</include>

Використання параметрів що передаються через атрибут locals в компоненті.

Приклад

<div class="block">
    <h2>{{title}}</h2>
    <p>{{description}}</p>
    <img src="{{img}}" alt="{{title}}">
</div>
