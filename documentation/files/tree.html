Структура проекту

Цей документ описує основну структуру проекту.

Головна HTML-компоненти Умови та умовні вирази Цикли та json Налаштування
...
|--| dist/ (Зібраний проект)
|--| src/ (папка з похідними файлами)
|-----| html/ (Файли HTML)
|--------| components/ (Компоненти)
|-----------| header/
|--------------| _header.html
|--------------| header.scss
|-----------| footer/
|--------------| _footer.html
|--------------| footer.scss
|-----------| card/
|--------------| _card.html
|--------------| card.scss
|--------------| cards.json
|--------| sections/ (Секції)
|-----------| hero/
|--------------| _hero.html
|--------------| hero.scss
|--------| modals/ (Попапи)
|-----------| auth-modal/
|--------------| _auth-modal.html
|--------------| auth-modal.scss
|--------| ui/ (UI елементи "кнопки, інпути, інше...")
|-----------| button/
|--------------| _button.html
|--------------| button.scss
|-----| js/ (Скрипти проекту)
|--------| main.js/ (Головний файл скриптів")
|--------| custom.js/ (Кастомні скрипти")
|-----| scss/ (Стилі)
|--------| base/ (Базові стилі")
|-----------| base.scss (Підключаються файли з папки)
|-----------| mixins.scss (Міксини)
|-----------| reset.scss (Обнулення)
|-----------| variables.scss (CSS змінні)
|--------| base/
|-----------| fonts.scss (Шрифти)
|-----------| icons.scss (Іконки іконочного шрифта)
|--------| libs/ (Стилі бібліотек "swiper, lightgallery і т.д.")
|--------| settings/
|-----------| media.scss (Розміри брекпоінтів)
|-----------| params.scss (Глобальні налаштування проекту "контейнер, полотно, шрифт і т.д.")
|--------| connect.scss Підключення всіх файлів для імпорту в main.scss
|--------| global.scss Глобальні стилі
|--------| tailwindcss.scss Імпорти та стилі для TailwindCSS
|--------| main.scss Головний файл проекту
|-----| assets/ (Файли проекту)
|--------| files/ (Різні файли "копіюються при білді")
|--------| fonts/ (Шрифти Woff2)
|--------| img/ (Зображення")
|--------| video/ (Відео")
...
