Цикли з масивами та Json файлом

Цей документ описує підключення компонентів в циклах, ввивід контенту з json файлу.

Головна Структура HTML-компоненти Умови та умовні вирази Налаштування

Використання контенту з json:

  • response - містить всього json файлу,
  • post (можна вказати свою назву) - містить один елемент json файлу,
  • post (можна вказати свою назву) - містить один елемент json файлу,
  • index - індекс елемента (за потреби можна використовувати)

Підключення компонента циклом з json

Приклад

<fetch url="./src/html/components/card/cards-data.json">
    <each loop="post, index in response">
        <include
            src="./src/html/components/card/_card.html"
            locals='{
                "title": "{{post.title}}",
                "description": "{{post.description}}",
                "img": "{{post.img}}"
            }'>
        </include>
    </each>
</fetch>

Вивід контента одразу циклом з json

Приклад

<fetch url="./src/html/components/card/cards-data.json">
    <each loop="post, index in response">
        <div class="card">
            <h2>{{post.title}}</h2>
            <p>{{post.description}}</p>
            <img src="{{post.img}}" alt="{{post.title}}">
        </div>
    </each>
</fetch>

Використання циклу для масиву що передається в параметрі list.

Варіант 1.

Приклад виклику

*В даному випадку параметри item nf index можна не передавати, так як вони вже передаються автоматично

<ul>
    <each loop="item, index in ['facebook', 'telegram', 'facebook','twitter']">
        <include src="./src/html/components/card/_list-item.html"></include>
    </each>
</ul>

Приклад виводу в компоненті

<li><span>{{index}}</span> {{item}}</li>

Варіант 2.

Приклад виклику

<include
    src="./src/html/components/card/_list.html"
    locals='{"list": ['facebook', 'telegram', 'facebook','twitter']}'>
</include>

Приклад виводу в компоненті

<ul>
    <each loop="item, index in list">
        <li>{{ index }}: {{ item }}</li>
    </each>
</ul>
