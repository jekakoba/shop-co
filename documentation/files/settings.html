Налаштування проекту

Цей документ описує підключення та використання TailwindCSS, налаштування оптимізації зображень та стилів.

Файл налаштувань збірки template.config.js в корні проекту

Головна Структура HTML-компоненти Умови та умовні вирази Цикли та json

Як вмикнути TailwindCSS в проекті:

потрібно вказати tailwindcss: true

export default {
    tailwindcss: true,
    cleanCss: true,
    images: {
        ...
    },
 }

В головному файлі стилів src/scss/main.scss в самому верху потрібно розкоментувати наступне

@import "tailwindcss";

Файл tailwindcss.scss використовується для імпорту стилів TailwindCSS та написання стилів компонентів TailwindCSS

Файл tailwind.config.js що в корні проекту, використовується для налаштування TailwindCSS

Очищення CSS в Build від не потрібних стилів:

потрібно вказати cleanCss: true

export default {
    tailwindcss: true,
    cleanCss: true,
    images: {
        ...
    },
 }

Оптимізація зображень в Build:

Для оптимізації в оригінальному форматі вказати imageMin: true.

Для створення WebP потрібно вказати imageMin: true та makeWebp: true. В ignore-webp масивом вказуються класи для яких не створювати WebP.

Для автоматичного заповнення атрибутів width та height вказати addImgSizes: true.

Для налаштування якості зображень вказати в imageQuality: { jpg: 80, png: [0.6, 0.8], webp: 80 }.

export default {
    ...
    images: {
        imageMin: true,
        makeWebp: true
        addImgSizes: true
        ignoreWebpClasses: ['ignore-webp'],
        imageQuality: {
            jpg: 80,
            png: [0.6, 0.8],
            webp: 80
        }
    },
}
