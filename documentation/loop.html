Cycles with arrays and json file

This document describes the connection of the components in the cycles, the content of the content from the JSON file.

Using Content with Json:

  • response - contains a total of json file,
  • post (You can specify your name) - contains one json file item,
  • post (You can specify your name) - contains one json file item,
  • index - item index (can you use)

Connecting the component cycle with JSON

Example

<fetch url="@с/card/cards-data.json">
    <each loop="post, index in response">
        <include src="@с/card/_card.html"</include>
    </each>
</fetch>

*When calling the component object post and index are transmitted automatically and available in the component as follows:

_card.html
<div>
    <h2>{{post.title}}</h2>
    <p>{{post.text}}</p>
    <span>{{post.date}}</span>
</div>

Content output immediately with a cycle with JSON

Example

<fetch url="@с/card/cards-data.json">
    <each loop="post, index in response">
        <div class="card">
            <h2>{{post.title}}</h2>
            <p>{{post.description}}</p>
            <img src="{{post.img}}" alt="{{post.title}}">
        </div>
    </each>
</fetch>

Using the array cycle transmitted in the parameter list.

An example of a call

<ul>

Version 2.

An example of a call

<include
    src="@с/card/_list.html"
    locals='{"list": ['facebook', 'telegram', 'facebook','twitter']}'>
</include>

An example of output in a component

<ul>

        <li>{{ index }}: {{ item }}</li>
    </each>
</ul>
